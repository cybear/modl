Welcome to the Malmö Open Device Lab

modl is basically an open mobile device test lab based on contributions.

Seeing Jeremy Keith's initiative, we felt that this is a brilliant idea. One that should be followed! One that we could set up in Malmö.

Add your devices

If you have a device you'd like to share:

  1. Fork the repo
  2. Add your device to devices.json
  3. Make a pull request!

Check out

$ cd your_repo_root/modl
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
